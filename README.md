## ✨ Features

### 🎨 Beautiful UI
- **VS Code-inspired design** with Activity Bar, Explorer, and Status Bar
- **3 stunning themes**: Nova Dark, Nova Light, Nova Midnight (OLED)
- **Custom vector icons** for every file type (Java, Python, JS, C++, Rust, Go, and more)
- **Smooth scrollbars** and modern tab design

### ️ Security First
- **Path Traversal protection** in Live Server
- **Command Injection prevention** in code execution
- **Safe file name validation** (blocks `..`, `/`, `\`, and other dangerous characters)
- **Localhost-only binding** for Live Server (no network exposure)
- **Restricted CORS** headers

### ⚡ Live Development
- **Built-in Live Server** with auto-reload for HTML/CSS/JS
- **Integrated Terminal** with command history
- **External Terminal support** (Windows Terminal, GNOME Terminal, xterm)
- **Real-time syntax highlighting** for 14+ languages

### 💻 Multi-Language Support
- ☕ Java | 🐍 Python | ⚡ JavaScript | 📘 TypeScript
- 🔷 C / C++ | 🦀 Rust | 🐹 Go | 🎯 Kotlin
- 💎 Ruby | 🐘 PHP | 📟 Bash | 🌙 Lua |  HTML/CSS

### 🛠️ Developer Tools
- **Line gutter** with current line highlighting
- **Smart indentation** (Tab = 4 spaces)
- **Find in editor** (Ctrl+F)
- **Custom keybindings** (Ctrl+Enter to run, Ctrl+S to save)
- **File Explorer** with context menu (New, Copy, Cut, Paste, Rename, Delete)
